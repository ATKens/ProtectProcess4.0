# 项目名称

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

ProtectProcess4.0

## 功能特性

- 特性1：进程保护防止在任务管理器中被关闭
- 特性2：当进程被强制关闭，3min后重启电脑

## 安装

1. 克隆项目到本地
2. 通过LoadDriverByDll2进行安装，直接调用即可自动安装
3. 安装依赖：`ProcessProtect install`
